# IVO Risk-Scan API - Produkt-skiss

## Vad:
API för sjukhus/vårdgivare att automatiskt granska dokumentation.

## Funktioner:
- Integrera IVO-riskbedömning i egna system
- JSON input → risk-rapport output
- Stödjer svenska vård-dokumentationsstandarder
- HIPAA/GDPR-kompatibelt

## Användningsområden:
- Efterjournalsystem
- Kvalitetssäkring
- IVO-förberedelse
- Utbildningssyfte
